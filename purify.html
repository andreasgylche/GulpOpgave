Gulp-purifycss

Denne gulp plugin analysere html og css filer og finder ud af hvilke classes og id'er som er brugt på hjemmesiden, og fjerner de css egenskaber som ikke bruges. Derudover korrigere den også css'en så det står pænt og læseligt.

Det er specielt godt når frameworks som Bootstrap bruges, hvilket er tilfældet i dette eksempel.

Den css fil for Bootstrap som bruges i dette tilfælde fylder 156 kilobites (billede til venstre) før purify kommandoen er kørt. Efterkommandoen fylder samme css fil nu 64 kilobites (billede til højre).
